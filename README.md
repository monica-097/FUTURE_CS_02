# Phishing Email Detection & Awareness System

## Overview
This project was completed as part of the Cyber Security Internship Task 2 by Future Interns.

The objective of this task was to analyze phishing email samples, identify phishing indicators, classify email risks, and create awareness guidelines to help users avoid phishing attacks.

---

## Objectives

- Analyze phishing email samples
- Investigate email headers
- Detect phishing indicators
- Classify email risks
- Create prevention and awareness guidelines

---

## Tools Used

- Google Admin Toolbox Message Header Analyzer
- MXToolbox Email Header Analyzer
- GitHub
- PDF Documentation Tools

---

## Key Phishing Indicators Identified

- Suspicious sender domain
- DKIM alignment failure
- Missing DMARC policy
- Fear and urgency-based language
- Third-party relay misuse
- Phishing impersonation techniques

---

## Risk Classification

| Indicator | Risk Level |
|-----------|------------|
| DKIM Alignment Failure | Critical |
| Hijacked Sender Domain | High |
| Panic/Urgency Language | High |
| Final Verdict | PHISHING |
